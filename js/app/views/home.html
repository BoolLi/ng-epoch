Ng-Epoch Simple charting for AngularJS

NG-Epoch is a set of directives wrapping the Epoch jQuery plugin. Install the directive and plugin, place a tag in your HTML, and feed it data. Look what made the following pie chart:

<epoch-pie chart-height="240" chart-width="240" chart-data="[{label:'A',value:10},{label:'B',value:20},{label:'C',value:40},{label:'D',value:30}]"></epoch-pie>

Install the module, inject a single dependency to your AngularJS app, place a single tag where you want the chart, and give it data. It all starts with:
bower install ng-epoch

A directive has been created for each chart and graph available in the original plugin. Every option is also fully supported plus a few extra bells and whistles to make things easier.

The code adds minimal weight to your app, but packs so much punch. Plus, your logic is easily separated making implementation and maintenance easy.
