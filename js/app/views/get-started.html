Step 1. Install

Run bower install ng-epoch from inside your project folder to install ng-epoch and its dependencies, which are AngularJS, jQuery, d3, and Epoch.

Step 2. Include

Important: When including the libraries in your page, ensure that D3 is loaded before Epoch.

In the head of your page, include the stylesheet that comes with Epoch. This will provide default colors and gauge styles. Instructions for customizing chart styles is available on the Epoch Project Site.

<link rel="stylesheet" href="[BOWER_PATH]/epoch/epoch.min.css" />

Next, add the following javascript sources to your page(s). Typically, sources are included in the <head> or just before the </body> in the HTML. In either case, if you do not already have jQuery and AngularJS included, add them:

<script src="[BOWER_PATH]/jquery/jquery.min.js"></script>
<script src="[BOWER_PATH]/angular/angular.min.js"></script>

At some point after those two libraries are included, add the following three files:

<script src="[BOWER_PATH]/d3/d3.min.js"></script>
<script src="[BOWER_PATH]/epoch/epoch.min.js"></script>
<script src="[BOWER_PATH]/ng-epoch/ng-epoch.js"></script>

Step 3. Implement

First, inject ng-epoch into your main AngularJS app like so: angular.module('YOUR_APP', ['ng.epoch']);

Then inside your page, place one of the <epoch-????> tags. For example, let's make a simple pie chart that is 160x160 and give it 2 values to draw.

<epoch-pie chart-height="160" chart-width="160" chart-data="[{label:'A',value:65},{label:'B',value:35}]"></epoch-pie>
